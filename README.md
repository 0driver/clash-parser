# clash-parser
# 直接将embed-clash-parser.js的文件内容复制到clash for windows的parser里面即可完成资源解析。
# 使用时，仍然需保证远程配置为clash格式。可以直接使用机场提供的clash原始订阅链接。不必再次通过订阅转换修改原始订阅里面的策略组，规则等配置。
# 使用parser后，将在本地解析配置文件，不存在隐私泄漏问题（但仍需保证远程配置文件为clash格式才能解析），可以在解析clash配置文件的同时，自定义策略组、分流规则等其他配置。
# 本项目中主要引用的是blackmatrix7中的rules。并包含热门节点（香港、台湾、日本、美国、新加坡、韩国）策略组。
